# Final-Year-Project
 This is a repository for my final year project "Using Large Language Models for text compression". It uses the Rank Zip algorithm along with the Arithmetic Coding + LLM PMF model algorithm as outlined in this paper https://arxiv.org/abs/2306.04050

## How to Run the Software

To run the software and verify the results, follow the steps below.

---

### 1. Download the Repository

First, clone or download the GitHub repository from:

https://github.com/panosdou/Final-Year-Project/

---

### 2. Set Up the Environment

It is recommended to run the software inside a virtual environment.

- The code was written in **Python 3.10.13**, so make sure this version is installed.
- Alternatively, you may use **conda** to create an environment and select Python 3.10.13 as the interpreter.

---

### 3. Install Dependencies

Install all required packages using `pip`:

```bash
pip install -r requirements.txt
